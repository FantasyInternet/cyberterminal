Keyboard shortcuts
------------------
Shortcut  | Action
--------- | ------
Esc       | Send break signal
Hold Esc  | Force shutdown, return to previous connection
F11       | Full screen
F12       | Developer console
Alt+G     | Switch to game input
Alt+T     | Switch to text input
